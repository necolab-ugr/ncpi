FAQ

Version 1.0

Which Python versions are supported?

Python 3.10 or higher.

Do I need NEST for all workflows?

No. NEST is only required for simulation examples that depend on it.

Can I run FieldPotential without optional dependencies?

No. FieldPotential workflows require optional NEURON/LFPy-related libraries.

How should I run NEST workflows on Windows?

Use WSL2 (Ubuntu) and install NEST in the WSL environment.

Where do I report issues?

Open an issue on the project repository.
