ncpi Web interface

The ncpi Web interface is designed to facilitate the use of ncpi library and data visualization.

Installation of web GUI

You'll need to activate the virtual environment with conda to run the application.


conda activate ncpi-env

Run the web application

Open a terminal window. Go to webui/ folder. Then, run the application with:


flask run

To access the frontend in the same computer you are running this web application, visit http://localhost:5000/ in your browser (or the URL flask indicates in the terminal window).

Use of ncpi Web interface

The Web interface is divided into 5 modules. You can use them individually or as a full pipeline.
