ncpi: Neural Circuit Parameter Inference

Version 1.0 Python >= 3.10

ncpi is a Python library for model-based inference of neural circuit parameters from population-level electrophysiological signals such as LFP, EEG, and MEG.

A unified simulation-inference workflow

One environment, from simulated neural circuits to real data

Imagine moving from a mechanistic hypothesis about a neural circuit to simulated population activity, multiscale field potentials, biophysically meaningful biomarkers, and trained inference models without assembling a different software toolchain for every step.

ncpi connects simulation, forward modelling, feature extraction, inference, prediction, and analysis in a modular and reproducible end-to-end workflow. Use only the components you need, or combine them into a complete pipeline that links computational models with empirical measurements across recording scales.

Simulate and observe

Generate synthetic neural activity and transform it into CDM, LFP, EEG, or MEG observables.

Extract and infer

Extract interpretable biomarkers, train inverse models, compute predictions, and analyze results.

Choose how you work

Build workflows in pure Python, develop interactively in Jupyter, or use the guided WebUI.

Run wherever research happens

Move reproducible workflows from local workstations to shared server infrastructure.

The intuitive WebUI turns complex modelling and data-processing pipelines into guided configurations while keeping scientifically relevant parameters accessible. Flexible data loading supports different empirical dataset structures, documented installation paths cover Linux, macOS, and Windows environments, and the toolbox and its scientific foundations have been published in peer-reviewed publications.
