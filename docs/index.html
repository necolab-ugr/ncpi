ncpi: Neural Circuit Parameter Inference

Version 1.0 Python >= 3.10 WebUI Included

ncpi is a Python library for model-based inference of neural circuit parameters from population-level electrophysiological signals such as LFP, ECoG, EEG, and MEG.

Highlights

  • Integrated simulation, forward modelling of field potentials, feature extraction, inference, and analysis pipeline.
  • Modular architecture so you can use only the modules you need.
  • A super nice graphical interface (WebUI) designed for practical end-to-end workflows.

Source repository: https://github.com/necolab-ugr/ncpi
