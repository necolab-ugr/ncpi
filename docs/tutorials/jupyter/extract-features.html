Feature Extraction

This tutorial provides a Jupyter notebook that loads CDM signals from the field-potential tutorial and computes feature vectors for each J_ext condition. It extracts all 22 catch22 features to capture complementary time-structure, statistical, and spectral-related aspects of biological signals, and computes specparam descriptors that separate aperiodic and oscillatory components, focusing on slope and gamma peak frequency. It then visualizes the 24 feature trends in a 6 x 4 subplot figure.

Notebook file: extract-features.ipynb
