Training an Inverse Model and Computing Predictions

This tutorial mirrors the ncpi README pipeline and explicitly combines both stages: training inverse models from simulation-derived features and computing predictions of the ground-truth E/I ratio. It also includes a side-by-side comparison of multiple machine-learning regressors.

Models compared in the notebook: RandomForestRegressor, Ridge, and MLPRegressor.

Notebook file: training-predictions.ipynb
