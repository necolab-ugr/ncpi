Tutorials for using the ncpi toolbox

Explore hands-on tutorials to get started with ncpi and build your own workflows. The Core Workflow guides you through the complete pipeline from neural simulation to parameter inference: simulating neural activity with biophysical models, extracting informative features from the data, and training inverse models to recover parameters from observations. The Target Audience Tutorials provide specialized guidance for experimentalists who want to connect measurements with model parameters, computational neuroscientists developing data-constrained models, and researchers in machine learning and simulation-based inference (SBI) working on neural systems.
