How custom simulation files are used

Upload Python scripts for network and simulation steps. The WebUI executes them in order and reads generated outputs.

Simulation Categories

Based on conventions in NEST simulations, ncpi.Simulation is organized in 3 categories: network (network specification), simulation (network creation + simulation run), and analysis (post-processing of results).

Execution Flow

  1. Run the network implementation script using values from the network parameters script.
  2. Run the simulation implementation script using values from the simulation parameters script.

In this WebUI, the analysis script is intentionally left out because the Analysis module already provides graphical analysis tools.

You have full freedom to design these scripts (single trial or parameter sweep with multiple trials). The only limitation is that simulation outputs must be saved as .pkl files so they can be loaded by subsequent WebUI modules.
